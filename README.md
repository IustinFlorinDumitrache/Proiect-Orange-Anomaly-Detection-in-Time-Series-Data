Anomaly detection framework - A scalable, LLM-driven anomaly detection framework engineered for real-time monitoring across Oasis, Pioneer, SMSC, MMSC, and VMS platforms—processing
over 500 operational metrics at one-minute granularity. Leveraging zero-shot learning capabilities, the framework identifies subtle, complex, and evolving anomalies in high-dimensional 
telemetry streams without requiring labeled training data, enabling proactive incident detection, reducing mean-time-to-detect (MTTD), and enhancing overall platforms reliability.
